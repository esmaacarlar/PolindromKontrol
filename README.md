# Polindrom Kontrolü

Bu proje, kullanıcının girdiği bir metnin polindrom olup olmadığını Java'da stack veri yapısını kullanarak test eder.
• Stack yapısının "Son giren ilk çıkar" prensibiyle karakterlerin ters çevrilmesi sağlanır
• Girilen string ifade yığıta atılır ve çıkarılırken orijinaliyle karşılaştırılır
